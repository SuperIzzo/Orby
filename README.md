Orby
=====================================

Made for Ludum Dare 35 (Theme: Shapeshift)

https://superizzo.itch.io/orby

![Orby Poster](https://img.itch.zone/aW1nLzM5MDQ4MjkucG5n/315x250%23c/wewNoO.png)

Orby is a small round creature which has the ability to transform itself into an invincible orb that defies gravity.

Use "left shift" or "numpad ." to orb.

It is a single (very hard) level.

This was jam submission for Ludum Dare 35 Compo (ages ago...)
